# ☕ Coffee Shop Sales Dashboard

## 📌 Project Overview
This project showcases an interactive **Coffee Shop Sales Dashboard** built using Microsoft Excel. It transforms raw transactional data into meaningful insights to help understand sales performance, product trends, and store comparisons.

---

## 🎯 Project Objectives
- Analyze sales performance, footfall, and order patterns across multiple stores
- Identify peak hours, top products, best-performing stores, and category trends
- Create a fully interactive dashboard using Microsoft Excel (PivotTables, charts & slicers)
- Deliver clear, actionable business insights and recommendations

---

## 🛠️ What I Built
- Fully interactive dashboard using **Pivot Tables**, **Pivot Charts**, and **Slicers**
- Sales analysis by store location (Astoria, Hell’s Kitchen, Lower Manhattan)
- Day-wise and category-wise performance trends
- A user-friendly layout for quick business insights

---

## ⚠️ Challanges & Solutions

**Challenges:**
- Raw, unstructured data with inconsistent formats
- Difficulty analyzing time-based trends
- Creating a clean, readable multi-chart dashboard
- Static visuals with no easy filtering

---

**Solutions:**
- Standardized dates and created derived columns
- Applied trimming and renaming for consistency
- Built dynamic PivotTables with slicers for Store, Day, and Category
- Designed a clean, consistent layout with unified color scheme
- Made the entire dashboard fully interactive


## 📊 Key Insights
- Coffee products dominate overall sales (**39%**)
- **Hell’s Kitchen** is the best performing store (**$236,511** sales and **50,735** footfall)
- Peak sales hours: **8 AM – 10 AM**
- **Barista Espresso** is the highest selling product (**$91,406**)
- Friday has the most orders however Saturday is the weakest day

---

## 💡 Business Recommendations
- Focus staffing and inventory during 8–10 AM rush
- Promote Barista Espresso and Coffee category
- Run weekend promotions to boost Saturday sales
- Reduce “Not Defined” size entries through better POS training
- Upsell to increase average bill per person
